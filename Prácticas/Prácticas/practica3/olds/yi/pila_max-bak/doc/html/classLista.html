Descripción detallada

template<class T>
class Lista< T >

T.D.A. Lista.

Una instancia c del tipo de dato abstracto lista sobre un dominio T es una sucesión finita de elementos del mismo. En una lista, las operaciones de inserción y borrado tienen lugar en cualquier posición, denominado. Una lista de longitud n la denotamos

  • <a1,a2,a3,..,an>

El espacio requerido para el almacenamiento es O(n), donde n es el número de elementos de la lista.

La lista contiene una celda cabecera que no contiene datos útiles y es solamente el punto de entrada a ésta. Una lista vacía tiene por tanto un elemento. Los elementos se acceden a traves de objetos de tipo Iterador, que apuntan a la celda anterior del elemento en cuestión.
