Descripción detallada

template<class T>
class VD< T >

T.D.A. Vector Dinámico.

Una instancia c del tipo de dato abstracto vector dinámico sobre un dominio T es una sucesión finita de elementos del mismo. En un vector, los elementos se encuentran contiguos en memoria por lo que las operaciones de búsqueda y modificación en bucle resultan más eficientes.

El espacio requerido para el almacenamiento es O(n), donde n es el número de elementos del vector.
