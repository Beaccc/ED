T.D.A. Pila_max.

Una instancia pila del tipo de datos abstracto Pila_max es un objeto que se compone de un un objeto tipo Cola (TDA Cola) donde este guardara en cada nodo una estructura elemento que guarda "elem" un entero y el "max" hasta esa posicion dentro de la pila.

Lo representamos:

Cola<elemento> pila;

Un ejemplo de su uso:

#include <iostream>
#include "pila_max.h"
using namespace std;
int main(){
Pila_max p;
int i;
for ( i=10; i>=0 ; i--)
p.Poner(i);
while (!p.Vacia() ){
elemento x = p.Tope();
cout << x<<endl;
p.Quitar();
}
return 0;
}
